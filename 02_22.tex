\par\noindent
Some review:
\begin{itemize}
	\item For $(X,\omega)$ a symplectic manifold, given $H\in{}C^{\infty}(X)$, $\exists\unique\Xi_{H}\in\mf{X}(X)$ \st{} $\iota_{\xi_{H}}\omega=-dH$. Refer to Lee, Smooth Manifolds, Chapter 22. Warning: His $\omega$ is different from ours by a sign. But the $\xi_{H}$ is the same.
	\item The flow of $\xi_{H}$ preserves $H$, $\omega$, $\frac{\omega^{n}}{n!}$, and the Liouville volume on regular level sets of $H$.
	\item Symmetries: We proved that $\forall{}G,H\in{}C^{\infty}(X)$, $\comm{\Xi_{G},\Xi_{H}}=0$ iff $\omega(\Xi_{G},\Xi_{H})=\Xi_{G}(H)=-\Xi_{H}(G)$ is constant. We call this the Poisson bracket, and denote it by $\omega(\Xi_{G},\Xi_{H})=\brace{G,H}$.
\end{itemize}

\defn{
	We say that $G$ is a \u{conserved quantity} under the flow of $\Xi_{H}$ if and only if $G$ is constant along the trajectories of $\Xi_{H}$ if and only if $\Xi_{H}(G)=0$.\n
}

\par\noindent
Observe: The above implies that $G$ is a conserved quantity under the flow of $\Xi_{H}$ if and only if $\brace{G,H}=0$, so $\Xi_{G}$ preserves $H$ if and only if $\Xi_{H}$ preserves $G$.\n

\par\noindent
We'll apply this as follows:
\begin{itemize}
	\item Geodesic flow is a Hamilton flow for some $H:T^{*}M\to\R$.
	\item If we have a field on $M$ generating isometries, we'll get a $G$ on $T^{*}M$ that preserves $H$. This implies that $G$ is constant along geodesics.
\end{itemize}

\par\noindent
Recall:
If $M$ is any $C^{\infty}$ manifold, then $X=T^{*}M$ has a natural symplectic form $\omega$, which, in standard local coordinates on $T^{*}M$, takes the form $\omega=dp_{i}\wedge{}dx^{i}=d\alpha$, where $\alpha=\sum{}p_{i}dx^{i}$ is the tautological $1$-form.\n

\par\noindent
A class of examples of Hamiltonians on $T{*}M$: Start with $X\in\mf{X}(M)$. Then define
\[
	\map{l_{X}:T^{*}M}{\R}{(x,\xi)}{\xi(X_{x})}
\]
Observe that $l_{X}(x,\xi)$ is linear in $\xi$, i.e., linear on the fibers.\n

\par\noindent
In coordinates $(x^{1},\ldots,x^{n})$, $X=f^{i}\pd{}{x^{i}}$, we get $(x^{1},\ldots,x^{n},p_{1},\ldots,p_{n})$ coordinates on $T^{*}U$. Then
\[
	l_{X}(x^{1},\ldots,x^{n},p_{1},\ldots,p_{n})=p_{i}f^{i}(X)
\]

\par\noindent
Recall Hamilton's equations for the flow of $\Xi_{l_{X}}\in\mf{X}(T^{*}M)$:
\[
	\left\{\begin{array}{ll}
		\dot{x}^{i}=\pd{l_{X}}{p_{i}}=f^{i} & \leftrightarrow\ptxt{exactly the flow of $X$ itself}\\
		\dot{p}^{i}=-\pd{l_{X}}{x^{i}}=-p_{j}\pd{f^{j}}{x^{i}}
	\end{array}\right.
\]

\prop{
	Let $\phi_{t}:M\to{}M$ be the flow of $X$. Then the flow of $\Xi_{l_{X}}$ is $\tilde\phi_{t}:T^{*}M\to{}T^{*}M$, given by
	\[
		\tilde\phi_{t}(x,\xi)=\paren{\phi_{t}(x),((d\phi_{-t})_{\phi_{t}(x)})^{*}\xi}
	\]
}

\par\noindent
Let's unpack this. We know
\[
	d(\phi_{-t})_{\phi_{t}(x)}:T_{\phi_{t}(x)}M\to{}T_{x}M
\]
Thus, its pullback is a map $T_{x}^{*}\to{}T_{\phi_{t}(x)}^{*}M$. So we have the following commutative diagram:
\[
	\begin{tikzcd}
		T^{*}M \arrow[d] \arrow[r,"\tilde\phi_{t}"] & T^{*}M \arrow[d]\\
		M \arrow[r,"\phi_{t}"] & M
	\end{tikzcd}
\]

\newpage
\par\noindent
Our main example is when $(M,g)$ is a Riemannian manifold. Using $g$, $\forall{}x\in{}M$, we get $T_{x}M\cong{}T_{x}^{*}M$ by $\mathbb{F}:v\mapsto\iprod{\cdot,v}$. We can assemble this into a ``big map'' between the total spaces of the bundle:
\[
	\begin{tikzcd}
		TM \arrow[dr] \arrow[rr, "\sim", "\mathbb{F}"'] &[-10pt] & [-10pt] T^{*}M \arrow[dl]\\
		& M
	\end{tikzcd}
\]
by computing $\mathbb{F}$ fiberwise. In coordinates, let $(x^{1},\ldots,x^{n})$ be coordinates on $U\osubset{}M$. Then we get coordinates on $TU$ and $T^{*}U$, and $\mathbb{F}$ is
\[
	\mathbb{F}(x^{1},\ldots,x^{n},v^{1},\ldots,v^{n})=(x^{1},\ldots,x^{n},p_{i}=g_{ij}v^{j})
\]

\thm{
	Let $\map{L:TM}{\R}{(x,v)}{\frac{1}{2}\norm{v}^{2}}$. Define $H:T^{*}M\to\R$ by $H=L\of\mathbb{F}\inv$.
	\[
		\begin{tikzcd}[ampersand replacement=\&]
			\&[-10pt] TM \arrow[dl,"L"] \arrow[dr] \arrow[rr,"\sim","\mathbb{F}"'] \&[-10pt] \&[-10pt] T^{*}M \arrow[dl] \arrow[ll,bend right=30, "\mathbb{F}\inv"'] \arrow[llld, bend right=90,"H=L\of\mathbb{F}\inv"']
			\\
			\R \& \& M
		\end{tikzcd}
	\]
	Then $\mathbb{F}$ intertwines the geodesic flow on $TM$ with the Hamilton flow of $H$.\n
}

\cor{
	Geodesic flow is volume preserving, and we can use the Hamiltonian to study symmetries and other things.\n
}

\par\noindent
We could prove this now, but it would require a terribly long and boring computation. There's an elegant proof of this using a different point of view, which we will do later on.\n

\par\noindent
Observe: $H(x,p)=\frac{1}{2}g^{ij}(x)p_{i}p_{j}$. $\dot{x}^{i}=\pd{H}{p_{i}}=g^{ij}(x)p_{j}$.\n

\par\noindent
Application: Surfaces of revolution. Let $S=\d_{\Theta}\in\mf{X}(M)$ generate rotations: $\phi_{t}:M\to{}M$ where $\phi_{t+2\pi}=\phi_{t}$, and $\forall{}t$, $\phi_{t}$ is an isometry. (Compare with problem 1 on page 77 of Do Carmo.)\n

\par\noindent
On the cotangent bundle, $\tilde\phi_{k}:T^{*}M\to{}T^{*}M$ preserves $H$; therefore, $l_{\d_{\Theta}}:T^{*}M\to{}T^{*}M$ is a conserved quantity. But what is it geometrically? We can pass it to $TM$:
\[
	\map{l_{\d_{\Theta}}\of{}\mathbb{F}:TM}{\R}{(x,v)}{\iprod{v,\d_{\Theta}}}
\]
where $\giprod$ is the Euclidean dot product (from the subspace-induced Riemannian metric). Again, let $\gamma(t)$ be a geodesic with speed $1$. We know that
\[
	\iprod{\dot\gamma(t),\d_{\Theta}}=\norm{\d_{\Theta}}\cos\angle(\dot\gamma(t),\d_{\Theta})
\]
is independent of $t$. It turns out $\norm{\d_{\Theta}}=r$, the distance to the axis of symmetry, since the line of latitude has perimeter $2\pi\norm{\d_{\Theta}}$. So we conclude that along a speed-$1$ geodesic, $\gamma(t)\cos\angle(\dot\gamma(t),\d_{\Theta})$ is independent of $t$.
